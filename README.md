# Bird App

A Kotlin Multiplatform app showcasing cross-platform development with shared business logic across Android and other platforms.

## Features

- Kotlin Multiplatform project structure
- Shared business logic
- Platform-specific UI implementations
- Modern Kotlin patterns and APIs

## Tech Stack

- **Language:** Kotlin
- **Framework:** Kotlin Multiplatform
- **UI:** Compose Multiplatform / Platform Native

## Getting Started

1. Clone the repository
```
git clone https://github.com/bhartendukodes/bird-app.git
```
2. Open in Android Studio or IntelliJ IDEA
3. Build and run the desired target
